# 📊 Market Risk Monitor

## 🚀 Overview
Market Risk Monitor is an advanced financial analytics project designed to evaluate portfolio performance, quantify risk, and simulate future outcomes using statistical modeling and data visualization.

The project integrates Python-based quantitative analysis with interactive Tableau dashboards to deliver actionable insights for investment decision-making.

---

## 🎯 Objectives
- Analyze portfolio returns and volatility
- Compute Value at Risk (VaR) for downside risk estimation
- Perform Monte Carlo simulations for future forecasting
- Visualize financial insights using Tableau dashboards
- Enable data-driven investment strategies

---

## 🧰 Tech Stack
- **Python** (Pandas, NumPy)
- **Data Processing** (CSV datasets)
- **Visualization** (Tableau Public)
- **Version Control** (Git, GitHub)
